# Руководство по безопасному использованию Alert Bot

## ⚠️ Важно понимать

Этот бот использует **userbot** (работает от личного аккаунта), что технически нарушает Terms of Service Telegram. Однако при правильном использовании риски минимальны.

---

## 🛡️ Оценка рисков

### Ваш случай использования:

✅ **Публичный канал** (https://t.me/razvozhaev)
✅ **Только чтение** (без записи от имени пользователя)
✅ **Низкая частота**: 30 секунд = 120 запросов/час
✅ **Один канал**, один бот

### Оценка риска бана: **< 1% при соблюдении рекомендаций**

## 🎯 Что делает бот безопасным

### 1. Встроенные защиты (УЖЕ ВКЛЮЧЕНЫ):

✅ **Случайные задержки** (25-35 сек вместо строго 30)
```env
RANDOM_DELAY=true
DELAY_VARIANCE=5
```

✅ **FloodWait обработка** - автоматически ждет, если Telegram попросит
```python
except FloodWaitError as e:
    wait_time = e.seconds
    logger.warning(f"Ждем {wait_time} секунд")
    await asyncio.sleep(wait_time)
```

✅ **Официальная библиотека Telethon** (не кастомная реализация)

✅ **Минимум действий** - только чтение и публикация в свой канал

## 📋 10 правил безопасного использования

### 1. ✅ ОБЯЗАТЕЛЬНО: Используйте отдельный аккаунт Telegram

**Не используйте основной аккаунт!**

**Создайте новый:**
1. Купите дешевую SIM-карту (~$1-5)
2. Зарегистрируйте новый Telegram аккаунт
3. Используйте этот аккаунт только для бота

**Почему это важно:**
- Если аккаунт забанят - потеряете только бота, не основной аккаунт
- Изолирует риски

---

### 2. ✅ Не меняйте CHECK_INTERVAL на меньше 20 секунд

```env
# ✅ Хорошо
CHECK_INTERVAL=30
CHECK_INTERVAL=60

# ⚠️ Допустимо, но рискованнее
CHECK_INTERVAL=20

# ❌ ОПАСНО - высокий риск FloodWait
CHECK_INTERVAL=10
CHECK_INTERVAL=5
```

**30 секунд - оптимально!**

---

### 3. ✅ Оставьте RANDOM_DELAY=true

```env
# ✅ ПРАВИЛЬНО
RANDOM_DELAY=true

# ❌ НЕ ДЕЛАЙТЕ ТАК
RANDOM_DELAY=false  # Слишком регулярно - выглядит как бот
```

Случайные задержки имитируют человеческое поведение.

---

### 4. ✅ Мониторьте ОДИН канал

```env
# ✅ ПРАВИЛЬНО
SOURCE_CHANNEL=@razvozhaev

# ❌ НЕ ЗАПУСКАЙТЕ несколько ботов с одного аккаунта
# Не мониторьте 10+ каналов одновременно
```

---

### 5. ✅ Используйте тот же IP/локацию, что и обычно

**Если ваш основной Telegram:**
- Используется без VPN → запускайте бота без VPN
- Используется с VPN → запускайте бота через тот же VPN

**Не меняйте IP часто** - это подозрительно для Telegram.

---

### 6. ✅ Не используйте бот для других задач

**Этот аккаунт = только для этого бота**

❌ **НЕ делайте с этого аккаунта:**
- Массовые рассылки
- Автоматический add в группы
- Парсинг участников
- Множественные боты

✅ **Только:**
- Чтение одного публичного канала
- Публикация в свой канал

---

### 7. ✅ Следите за логами

```bash
tail -f logs/bot_*.log
```

**Обращайте внимание на:**
- `FloodWaitError` - если часто (>1 раз в день) - увеличьте CHECK_INTERVAL
- Любые ошибки авторизации
- Неожиданные отключения

---

### 8. ✅ Запускайте бота на стабильном VPS

**Хорошо:**
- Один VPS с постоянным IP
- Стабильное соединение
- Uptime 99%+

**Плохо:**
- Постоянные переподключения
- Разные IP каждый раз
- Домашний интернет с нестабильным соединением

---

### 9. ✅ Дайте аккаунту "прогреться"

**Если создали НОВЫЙ аккаунт:**

1. **День 1-2**: Просто используйте как обычный Telegram
   - Напишите пару сообщений
   - Подпишитесь на каналы
   - Поставьте аватар

2. **День 3-4**: Подпишитесь на целевой канал (@razvozhaev)
   - Полистайте его вручную

3. **День 5+**: Запускайте бота

**Почему:** Новые аккаунты с сразу активным API - подозрительны.

---

### 10. ✅ Проверяйте работу бота

```bash
# Каждые 1-2 дня проверяйте
sudo systemctl status alert_bot

# Проверяйте логи на FloodWait
grep -i "floodwait" logs/bot_*.log
```

## 🚨 Признаки проблем

### Когда СРОЧНО остановить бота:

❌ **FloodWaitError каждые 10-30 минут**
```
FloodWaitError: ждем 300 секунд
```
→ Увеличьте CHECK_INTERVAL до 60-120 секунд

❌ **Ошибки авторизации**
```
AuthKeyUnregisteredError
SessionRevokedError
```
→ Возможно, аккаунт забанен. Остановите бота.

❌ **Не работает без ошибок**
→ Возможно, проблема с конфигурацией

## 🆘 Что делать если аккаунт забанили

1. **Не паникуйте** - это временный бан (обычно 24-48 часов)

2. **Подождите** - попытки обойти бан усугубят ситуацию

3. **Не создавайте сразу новый аккаунт** с того же номера

4. **Если это был отдельный аккаунт** - вы ничего не потеряли!

5. **Обратитесь в поддержку** (если считаете бан ошибочным):
   - Напишите на @SpamBot
   - Объясните, что читали публичный канал
